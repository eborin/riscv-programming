RISC-V ALE Exercise Book

Overview

RISC-V ALE Exercise Book is a set of complementary exercises for the textbook An Introduction to Assembly Programming with RISC-V Its exercises cover topics from using the necessary tools to generating files and using the ALE simulator to writing assembly code to interect with external devices.

Most of the exercises were made to be executed on the RISC-V Assembly Learn Environment.

Target audience

This book is intended for people that are studying RISC-V programming and want to practice assembly programming.
